# Part 3: AI 서비스 통합

Part 1에서 배운 AI 원리와 Part 2에서 배운 데이터 분석을 조합해 실제 서비스를 만듭니다. 코드 리뷰를 예시로 사용하지만, 같은 패턴으로 문서 분석, 번역, 챗봇 등 다양한 AI 서비스를 구축할 수 있습니다.

## 전체 흐름

```
API → 프롬프트 → LLM → 파싱 → 액션 → 피드백
```

```
[분석 결과 + 데이터]
     ↓
[STEP 13] API 호출: AI에게 요청
     ↓
[STEP 14] 프롬프트: 효과적으로 질문
     ↓
[STEP 15] 모델 라우팅: 적합한 모델 선택
     ↓
[STEP 16] 응답 파싱: 답변에서 정보 추출
     ↓
[STEP 17] 액션 실행: AI 제안 실행
     ↓
[STEP 18] 피드백: 검증 및 반복
     ↓
[완성된 AI 서비스]
```
